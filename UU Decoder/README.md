# Uuencoding Format

How can you tell if data has been encoded using Uuencoding? 
Uuencoded data always starts with a "begin" line and terminates with an "end" line.

## Example: <br />
<b>begin</b> *mode filename* <br />
....encoded data.... <br />
Empty Line <br />
<b>end</b>
